How to make speech assistant with Python

The following are the stages of making a speech assistant program using the Python programming language jupyter notebook.

1. Import the required libraries

import speech_recognition as sr
import pyttsx3
import datetime
import pywhatkit
import wikipedia
Import speech_recognition is used to recognize the sound uttered by the user. Import pyttsx3 used to convert text to voice. imported datetime is used to retrieve the time parameter. Import pywhatkit is a library that has a lot function. In this speech assistant program, import pywhatkit is used to continue the system to access youtube and google. And finally import wikipedia used to retrieve information from wikipedia.

2. Use of speech recognition and pyttsx3 functions
listener = sr.Recognizer()
assistant = pyttsx3.init()
voices = assistant.getProperty('voices')
assistant.setProperty('voices', voices[1].id)
At this stage, the speech function begins to be used recognition to recognize the sound in the listener variable. Then the assistant variable is initialized text to speech so that the system can read text or text. In the voices variable, the process begins for accepts voice input and displays the sound output.

3. Create def talk(text)

def talk(text):
        pre assistant.say(text)
        assistant.runAndWait()

At this stage the assistant will say or output the sound according to the text and perform run and wait process.

4. Create def take_command()

def take_command():

        try:
                with sr.Microphone() as source:
                print('Device is listening, please speak...')
                voice = listener.listen(source)
                command = listener.recognize_google(voice)
                command = command.lower()
        except:
                pass
        return command

At this stage it is used to display the output "Device is listening, please speak..." when the program is run. At that point the system is ready to hear sound from user to then do an introduction to the sound

5. Create def run_assistant()

def run_assistant():
        command = take_command()

        if 'hello' in command:
                print('Hello Anggun, Can i help you')
                talk('Hello Anggun, Can i help you')
                run_assistant()
        elif 'time' in command:
                time = datetime.datetime.now().strftime('%I:%M %p')
                print('Current time is ' + time)
                talk('Current time is ' + time)
                run_assistant()
        elif 'love you' in command:
                print('Love you too and have a nice day, Anggun')
                talk('Love you too and have a nice day, Anggun')
        elif 'play' in command:
                song = command.replace('play', '')
                talk('playing ' + song)
                pywhatkit.playonyt(song)
        elif 'tell me about' in command:
                wiki = command.replace('tell me about', '')
                info = wikipedia.summary(wiki, 2)
                print(info)
                talk(info)
                run_assistant()
        elif 'thank you' in command:
                print('You are welcome and have a nice day, Anggun')
                talk('You are welcome and have a nice day, Anggun')
        else:
                talk('sorry i didnot get your question, i can search it from google')
                pywhatkit.search(command)
